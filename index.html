Introduction

Git is a distributed version control system that allows multiple developers to collaborate on a project without overwriting each other’s work. GitHub is a platform built around Git that enables users to host and share code, track issues, and collaborate on open-source projects.

Setup and Installation

Prerequisites

  • Install Git
  • Create a GitHub account

Clone this Repository

git clone https://github.com/your-username/learning-git-and-github.git

Replace your-username with your GitHub username.

Navigate into the Project Directory

cd learning-git-and-github

Git Basics

Cloning a Repository

git clone <repository-url>

Creating and Switching Branches

git checkout -b <branch-name>
git checkout <branch-name>

Committing Changes

git add <file-name>
git commit -m "Commit message"

Pushing Changes

git push origin <branch-name>

GitHub Basics

Creating a GitHub Repository

  • Go to GitHub
  • Click the "New" button to create a new repository
  • Follow the prompts to name and configure your repository

Forking and Pull Requests

Fork a repository to make your own copy, make changes, and submit a pull request to propose those changes.
